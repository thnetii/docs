CopyOrSpan<T>(ref T, out Boolean)

Creates a writable single-item span of the specified reference; or, if creating spans over reference is not supported by the runtime, copies the specified value to heap memory.

Declaration
public static Span<T> CopyOrSpan<T>(ref T reference, out bool isCopy)
Parameters
Type Name Description
T reference

A reference to the value to span over. If unsuccessful, the value will be copied to the heap instead.
Boolean isCopy

On return receives value that indicates whether the returned span is a heap-copy or provides direct access to reference.
Returns
Type Description
Span<T>

A Span<T> value with a length of 1.

If isCopy is true, the span wraps over a copy of reference which resides on the heap.

If isCopy is false, the span wraps directly over the specified reference reference.
