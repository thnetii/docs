Write(ref Int64, Int64)

Sets the bits as definied by this definition in the specified storage value, preserving the previous bit setting of all other bits.

Declaration
public long Write(ref long storage, long value)
Parameters
Type Name Description
Int64 storage

A reference to the storage value to write to.
Int64 value

The bit pattern to set.
Returns
Type Description
Int64

The resulting value of storage of the operation has completed.

(storage & InverseMask) | ((value << ShiftAmount) | Mask)
Remarks

value is first shifted upwards (to the left) to move it into the position where it can OR-ed with storage. In order to prevent setting additional bits, the shifted value is AND-ed with Mask.

The value of storage is AND-ed with InverseMask to clear all writable bits. The result is then OR-ed with the shifted masked value and becomes the new value of storage.

Since this operation is performed at bit-level, all logical intructions treat the operands as unsigned values. E.g. the shift is performed as a logical shift rather than an arithmetric one.
