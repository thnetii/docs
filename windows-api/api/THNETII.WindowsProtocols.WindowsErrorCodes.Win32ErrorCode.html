IsApplicationDefined

Whether the error code is an application-defined error code.

Declaration
public bool IsApplicationDefined { get; }
Property Value
Type Description
Boolean

true if the error code is application-defined; otherwise the error code is a system error code and the property value is false.
Remarks

Bit 29 is reserved for application-defined error codes.

No system error code has this property set to true.

If you are defining an error code for your application, set this bit to one. That indicates that the error code has been defined by an application, and ensures that your error code does not conflict with any error codes defined by the system.
