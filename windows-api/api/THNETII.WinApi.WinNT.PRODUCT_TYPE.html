Remarks

This list grows with each OS release.

There is no ordering of values to ensure callers do an equality test i.e. greater-than and less-than comparisons are not useful.

NOTE: Values in this list should never be deleted. When a product-type 'X' gets dropped from a OS release onwards, the value of 'X' continues to be used in the mapping table of .
