Type Description
Boolean

true if the current instance has a value and that value is equal to otherValue; otherwise, false.
Remarks

Equality between a Maybe<T> and a value of type T is commutative and determined by satisfying all of the following criteria:

  1. The HasValue of the Maybe<T> value returns true.
  2. The Value property of the Maybe<T> value is equal to the value of type T.

If T implements IEquatable<T>, Value is compared using the Equals(T) method. Otherwise, Equals(Object) is used.
