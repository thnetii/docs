Class CopyIOStream

A wrapper Stream implementation around another Stream instance that supports automatic copying of read and written data into a secondary Copy-Stream.

Remarks

Instances derived from CopyIOStream by default do not support seeking in order to keep the read and write copies consistent. However, if the source stream is seekable you can call Seek(Int64, SeekOrigin) on the BaseStream member. Note that doing so can potentially create garbled or discontinuous content in the read and/or write copies on subsequent I/O operations.

Exceptions
Type Condition
NotSupportedException

Instances of CopyIOStream do not support seeking.

Remarks

Instances derived from CopyIOStream by default do not support setting the length. However, if the underlying streams support it, you can call SetLength(Int64) individually on the BaseStream, ReadCopy and WriteCopy members. Note that doing so must take into account that SetLength(Int64) might not succeed on all three members.
