Class ConsoleColorContext

Contains the state of current and original Console foreground and background color settings.

The context should be used in a using-block to ensure proper resetting of console colors.

Examples
using System;
using THNETII.Common.Cli;

class Program
{
    static void Main(string[] args)
    {
        Console.WriteLine("This text is written normally.");
        using (var context = new ConsoleColorContext(ConsoleColor.Red))
        {
            Console.WriteLine("This text is written using red foreground coloring.");
        }
        Console.WriteLine("This text is written normally again.");
    }
}

Nullable<ConsoleColor> fgColorOriginal

The original foregorund console color to reset to when the context is disposed, or null if the foreground color should not be reset.

The original background console color to reset to when the context is disposed, or null if the background color should not be reset.
