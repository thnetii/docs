ConvertedValue

Gets or sets the Converted value that represents the conversion from RawValue to the TConvert type.

Declaration
public TConvert ConvertedValue { get; set; }
Property Value
Type Description
TConvert

The result of converting RawValue from the TRaw type to the TConvert type.
Remarks

Both concurrent set and get accesses to ConvertedValue can be considered thread-safe.

The conversion from RawValue is defined by the argument to the constructor of the DuplexConversionTuple<TRaw, TConvert> class.

The conversion to RawValue is defined by the argument to the constructor of the DuplexConversionTuple<TRaw, TConvert> class.

The DuplexConversionTuple<TRaw, TConvert> class caches the result of the conversion. The conversion function is only called if RawValue has changed since ConvertedValue was last accessed.

If neither the value of RawValue nor ConvertedValue are changed, subsequent get accesses to ConvertedValue will return the exact same value or instance of TConvert.

If the value of ConvertedValue is changed by setting a new value, it is immediately converted to TRaw and RawValue is set to the result of the conversion.
