ConversionTuple(Func<TRaw, TConvert>, IEqualityComparer<TRaw>)

Creates a new conversion tuple with the specified conversion function.

Optionally, an EqualityComparer can be specified that is used to determine whether the raw value of the conversion has changed.

Declaration
public ConversionTuple(Func<TRaw, TConvert> rawConvert, IEqualityComparer<TRaw> rawEqualityComparer = null)
Parameters
Type Name Description
Func<TRaw, TConvert> rawConvert

The conversion function to use, to convert values from TRaw to TConvert. Must not be null.
IEqualityComparer<TRaw> rawEqualityComparer

An optional equality comparer, to check whether the cached value of RawValue has been changed. Omit or specify null to use default equality checks.
Remarks

The conversion function rawConvert is only invoked when accessing ConvertedValue and then only if the value of RawValue has changed since the last access to ConvertedValue.

To determine whether RawValue has changed, the ConversionTuple<TRaw, TConvert> class caches the TRaw value that was used in the last conversion and compares it for equality against the current value of RawValue.

If no custom IEqualityComparer<T> is specified in rawEqualityComparer or if rawEqualityComparer is null, a reference equality check (ReferenceEquals(Object, Object) is used if TRaw is a reference type. If TRaw is a value type, Default is used as the equality comparer.

ConvertedValue

Gets the Converted value from the current RawValue.

Declaration
public TConvert ConvertedValue { get; }
Property Value
Type Description
TConvert

The result of converting RawValue from TRaw to TConvert.
Remarks

Concurrent accesses to ConvertedValue can be considered thread-safe.

The conversion from RawValue is defined by the argument to the constructor of the ConversionTuple<TRaw, TConvert> class.

The ConversionTuple<TRaw, TConvert> class caches the result of the conversion. The conversion function is only called if RawValue has changed since ConvertedValue was last accessed.

If the value of RawValue is not changed, subsequent accesses to ConvertedValue will return the exact same value or instance of TConvert.
