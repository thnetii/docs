Class BooleanStringConverter

Provides conversion methods to convert strings to a boolean value.

Remarks

A String value can be converted to a Boolean value according to the following rules (in listed order)

  1. Using the TryParse(String, Boolean) method.
  2. null is converted to false.
  3. Using the TryParse(String, NumberStyles, IFormatProvider, Int32) using a NumberStyles value of Any. Non-zero converted integer values convert to true, 0 (zero) converts to false.
  4. Using the TryParse(String, NumberStyles, IFormatProvider, Int32) using a NumberStyles value of HexNumber. The string must be prefixed with "0x" (case-insensitive). Non-zero converted integer values convert to true, 0 (zero) converts to false.
  5. If the string "yes" starts with the input, the input is converted to true (case-insensitive). E.g. "yes" and "y".
  6. If the string "no" starts with the input, the input is converted to false (case-insensitive). E.g. "no" and "n".
All inputs may be pre- or suffixed with white-space characters. The input string is trimmed if necessary. Empty or white-space-only strings will NOT be converted to Boolean values.

Boolean values are converted to String values using the ToString() method.
