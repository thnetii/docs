Parse<T>(String)

Converts the string representation of the constant name, serialization name or the numeric value of one or more enumerated constants to an equivalent enumerated value of T.

This operation is always case-insensitive using ordinal string comparison.

Declaration
public static T Parse<T>(string s)
    where T : struct, Enum
Parameters

Returns
Type Description

Type Parameters
Name Description
T

The enumeration type to convert to.
Remarks

The serialization name refers to the value specified in for the Name member of an XmlEnumAttribute applied to one of the enumerated constants of the T enumeration type.
