Enum JsonWebKeyType

Identifies the cryptographic algorithm family used with the key, such as "RSA" or "EC". Well-known JSON Web Key Type values should be registered in the IANA "JSON Web Key Types" registry established by the the JSON Web Algorithms RFC [RFC7518]

Namespace: THNETII.Security.JOSE

Fields

Name Description
Ec

JWKs can represent Elliptic Curve [DSS] keys. In this case, the "kty" member value is "EC".
Oct

When the JWK "kty" member value is "oct" (octet sequence), the JWK is used to represent a symmetric key (or another key whose value is a single octet sequence).
Rsa

JWKs can represent RSA [RFC3447] keys. In this case, the "kty" member value is "RSA". The semantics of the parameters defined for the JSON Web Key are the same as those defined in Sections 3.1 and 3.2 of RFC 3447.
Unknown

An unrecognized JSON Web Key Type.
