public string PayloadBase64UrlString { get; set; }
Property Value
Type Description
String
Remarks

The "payload" member MUST be present and contain the JWS Payload as a Base64 URL-safe encoded string.

Setting this property will also affect the value of the Payload property.

The value of this property is part of the signing input when computing the Signature included in the JWS.

Declaration

Returns
Type Description

A Base64 URL-safe encoded string, with the ProtectedBase64UrlString, the PayloadBase64UrlString and the SignatureBase64UrlString each separated with a period ('.') character.
