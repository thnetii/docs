ReadAsStreamReaderAsync(HttpContent)

Serialize the HTTP content and return an appropiate stream reader instance for reading the content as an asynchronous operation.

Declaration
public static Task<StreamReader> ReadAsStreamReaderAsync(this HttpContent httpContent)
Parameters
Type Name Description
HttpContent httpContent

The HTTP content to read.
Returns
Type Description
Task<StreamReader>

The task object representing the asynchronous operation.
Remarks

This method evaluates the ContentType header of httpContent and tries to create a StreamReader instance that uses the Encoding specified by the CharSet value of the content-type.

Creating a StreamReader with an approiate Encoding is best effort, if the HTTP Content does not contain any charset information, a StreamReader instance is created using the default constructor.

Exceptions
Type Condition
ArgumentNullException

httpContent is null.
