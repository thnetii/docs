EditDistance(String, String)

Returns the edit distance from the current string a to the other string b.

The edit distance between two strings, is the number of character edit operations needed in order to change one string to another.

Declaration
public static int EditDistance(this string a, string b)
Parameters
Type Name Description
String a

The original string.
String b

The other string to calculate the distance to.
Returns
Type Description
Int32

The number of characters that need to be changed, inserted or deleted in a to produce b.
Remarks

This uses the Levenshtein distance algorithm with only a linear memory overhead.

This implementation is highly optimized!

The original implementation is taken from the editDistance implementation found in the standard library for the Nim programming language.
