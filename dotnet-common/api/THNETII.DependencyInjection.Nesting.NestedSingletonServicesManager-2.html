Class NestedSingletonServicesManager<TFamily, TKey>

A singleton dependency injection service that manages nested service provider instances under a root service provider.

This class should be used in conjunction with the NestedScopedServicesManager<TFamily, TKey> to properly enforce nested scoping boundaries.
